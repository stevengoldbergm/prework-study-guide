HTML

  • The head element contains information about the webpage itself.
    • This includes metadata and associated tags, as well as details important for Search-Enging Optimization (SEO).
    • The head element is not visible to the end-user.
    • When using an external CSS style sheet, the link must be placed in the 'head' element.

  • The body element represents the visible content shown to the user, including such items as:
    • Headers
    • Paragraphs of text
    • Images, and
    • External links

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Number values for both margin and padding address the top/bottom of the element first, followed by the left/right of the element.
    • E.g. Adding the code "padding: 0 5px" to your style sheet will result in the left and right sides of the element have a 5 pixel internal buffer, while the top and bottom edges have no buffer at all.

Git

  • Basic Git Commands
    • git -h: Brings up the help screen for git
    • git status: Checks the branch you are currently working in and identifies any untracked files.
    • git branch: Shows all available branches.
    • git branch [branchname]: Creates a new branch without changing your current branch
    • git branch -d [branchname]: Deletes an existing branch.
    • git checkout [branchname]: Allows you to checkout the specific existing branch
    • git checkout -b 'branchname': Creates a new branch off the current branch and checks it out.
